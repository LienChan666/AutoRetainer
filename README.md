# AutoRetainer 简体中文汉化版

本仓库是 [PunishXIV/AutoRetainer](https://github.com/PunishXIV/AutoRetainer) 的简体中文汉化维护分支。

## 分支说明

- `master`：当前维护分支，跟随上游更新；每次推送后都会自动构建并发布插件。
- `deprecated`：旧版历史分支，仅供查看，不再同步、构建或发布。
